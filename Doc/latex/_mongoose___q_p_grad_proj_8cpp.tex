Gradient projection algorithm. 

\begin{DoxyAuthor}{Author}
S. Nuri Yeralan
\end{DoxyAuthor}
Apply gradient projection algorithm to the quadratic program which arises in graph partitioning\+:

min (1-\/x)\textquotesingle{}(D+A)x subject to lo $<$= b $<$= hi, a\textquotesingle{}x = b, 0 $<$= x $<$= 1

The gradient at the current point is provided as input, and the gradient is updated in each iteration.
